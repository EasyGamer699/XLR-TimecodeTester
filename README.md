# XLR-TimecodeTester

### 🚧 This is Work in Progress.... 🚧

A compact device for testing and verifying timecode signals transmitted over XLR. It detects and analyzes LTC signals to ensure stable synchronization, making it ideal for live, studio, and broadcast environments.

## ⚙️ Features

- ✅ Tests and verifies LTC timecode over XLR
- ✅ Detects presence and signal stability
- ✅ Displays current timecode (HH:MM:SS:FF)
- ✅ Portable and compact design
- ✅ Helps diagnose sync and signal issues
- ✅ Plug-and-play operation
